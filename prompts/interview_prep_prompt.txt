------------------------------------------------------------
Create backend/app/services/interview_qa_generator.py

This makes ONE Anthropic Claude call per session (cached — never called again for same session).
Check if InterviewQA rows already exist for the session_id before calling the API.

The single Claude prompt must generate all question categories in one request:

PROMPT TEMPLATE (keep tight):
"""
Generate interview questions and strong answers for this candidate.

JOB: {job_title} at {company_name}
JOB DESCRIPTION (truncated to 800 chars): {jd_excerpt}
CANDIDATE RESUME (truncated to 600 chars): {resume_excerpt}

Return JSON only. No preamble. Schema:
[
  {
    "category": "technical|behavioral|situational|role-specific",
    "question": "...",
    "answer": "..."
  }
]

Generate 5 technical, 4 behavioral, 3 situational, 3 role-specific questions.
Answers must reference the candidate's actual experience where possible.
Be specific, not generic. No buzzwords.
"""

Parse the JSON response. Bulk-insert all InterviewQA rows in one DB operation.
Return list[InterviewQA].

TASK B4 — Company Links Enrichment Service
------------------------------------------------------------
Create backend/app/services/company_enrichment.py

Given a company name, attempt to find its key URLs using a simple web search
via the existing httpx client (use DuckDuckGo instant answer API or a search
query — do NOT add a paid search API):

  GET https://api.duckduckgo.com/?q={company_name}+careers&format=json&no_redirect=1

Extract AbstractURL as the likely company website.
For the other URLs (Glassdoor, LinkedIn, Indeed, careers page), construct best-guess
URLs from known patterns:
  LinkedIn:  https://www.linkedin.com/company/{slug}
  Glassdoor: https://www.glassdoor.com/Reviews/{slug}-reviews.htm
  Indeed:    https://www.indeed.com/cmp/{slug}

Where slug = company name lowercased, spaces replaced with hyphens.
These are best-guess links — label them clearly in the frontend as "may require search."
Return a CompanyLinks dataclass with all fields Optional[str].
Do not call any paid API. Do not raise on failure — return whatever was found.

TASK B6 — API Routes
------------------------------------------------------------
Create backend/app/api/routes/interview.py

Register it in the main router. Endpoints:

POST   /api/interview/session
       Body: job_title, job_description, company_name, resume_text, salary_info?,
             location?, seniority?
       Creates InterviewSession, triggers company enrichment and Q&A generation
       (Q&A generation is async background task — return session immediately,
        Q&A populates in background)
       Returns: full session object with company links

GET    /api/interview/session/{session_id}
       Returns full session with company links

GET    /api/interview/session/{session_id}/qa
       Returns all InterviewQA rows grouped by category

GET    /api/interview/session/{session_id}/resume.pdf
       Returns FileResponse of the stored PDF

GET    /api/interview/sessions
       Returns list of all sessions (id, job_title, company_name, created_at)
       For sidebar/history

DELETE /api/interview/session/{session_id}
       Deletes session and all related rows (cascade)

POST   /api/interview/{session_id}/chat
       Body: message (user's new message)
       Loads last 10 messages from DB as context (not full history — token efficiency)
       Calls groq_chat with compact system prompt:
         "You are an interview coach. Help the candidate prepare for {job_title} at
          {company_name}. Be specific and concise."
       Saves both user message and assistant response to DB
       Returns: assistant response string

GET    /api/interview/{session_id}/chat
       Returns full chat history ordered by created_at

DELETE /api/interview/{session_id}/chat/{message_id}
       Deletes one message

DELETE /api/interview/{session_id}/chat
       Clears entire chat history for this session

POST   /api/interview/{session_id}/brainstorm/thread
       Body: title
       Creates new BrainstormThread
       Returns thread object

GET    /api/interview/{session_id}/brainstorm/threads
       Returns all threads for session

PATCH  /api/interview/{session_id}/brainstorm/thread/{thread_id}
       Body: title
       Renames thread

DELETE /api/interview/{session_id}/brainstorm/thread/{thread_id}
       Deletes thread and all its messages

POST   /api/interview/{session_id}/brainstorm/thread/{thread_id}/message
       Body: message
       Same pattern as interview chat — last 10 messages as context
       System prompt: "You are a career strategist helping someone prepare for
                       {job_title} at {company_name}. Think creatively and practically."
       Saves and returns assistant response

GET    /api/interview/{session_id}/brainstorm/thread/{thread_id}/messages
       Returns all messages for thread
